# AWS CI/CD Pipeline with Jenkins

A simple CI/CD pipeline using Jenkins on AWS EC2 to automatically build and deploy code from a GitHub repository.

## Architecture

```
Developer (VS Code)
        │
    git push
        │
        ▼
     GitHub Repo
        │
        ▼
 Jenkins (AWS EC2)
        │
   Build & Deploy
        │
        ▼
   Web Application
```

## Technologies
- AWS EC2
- Jenkins
- Git & GitHub
- HTML, CSS, JavaScript
